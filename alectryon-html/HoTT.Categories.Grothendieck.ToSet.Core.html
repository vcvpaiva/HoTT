  Context `{Funext}.

  (**
     Quoting Wikipedia:

     The Grothendieck construction is an auxiliary construction used
     in the mathematical field of category theory.

     Let

<<
     F : C ‚Üí Set
>>

     be a functor from any small category to the category of sets.
     The Grothendieck construct for [F] is the category [Œì F] whose
     objects are pairs [(c, x)], where [c : C] is an object and [x : F
     c] is an element, and for which the set [Hom (Œì F) (c1, x1) (c2,
     x2)] is the set of morphisms [f : c1 ‚Üí c2] in [C] such that
     [F‚ÇÅ f x1 = x2].  *)

  Variable C : PreCategory.
  Variable F : Functor C set_cat.

  Record Pair :=
    {
      c : C;
      x : F c
    }.

  
H:Funext
C:PreCategory
F:Functor C set_cat
{c : C & (F _0 c)%object} <~> Pair

F:Functor C set_cat

    issig.
  Defined.

  Notation morphism s d :=
    { f : morphism C s.(c) d.(c)
    | F _1 f s.(x) = d.(x) }.
